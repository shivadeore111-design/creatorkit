# CreatorKit

CreatorKit is a full-stack Next.js web app for Instagram creators. It includes:

- AI Carousel Generator (Gemini 1.5 Flash)
- Square Fit Tool
- Image Splitter
- Photo Editor with face-api.js integration

## Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000/home`.

## Environment

Create `.env.local`:

```bash
GEMINI_API_KEY=your_key_here
```

## Deployment

Deploy directly on Vercel free plan.
